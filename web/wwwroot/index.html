Laevitas Skew Explorer options · skew · factor

Fetches option-skew data from the Laevitas API and computes WeightedSum, Factor, SMA30 and EMA30. Web wrapper around the standalone .NET client.

Preconfigured server-side key — pick a token and either fetch a date range or load the full history since inception. Data is cached on the server (SQLite), so charts reload instantly. Rolling metrics (Factor / SMA30 / EMA30) use a 60-day warm-up.
Scroll / pinch to zoom · drag to pan · click a legend item to toggle a series
