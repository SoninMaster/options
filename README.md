# OPTIONS

Nowy projekt. Pierwsza strona to czysty `index.html` — bez frameworka, bez buildu.

## Lokalny podgląd

```bash
python3 -m http.server 0 --bind 0.0.0.0
```

(albo dowolny inny serwer statyczny — `npx serve .`).

## Stack

Na razie zero zależności. Wszystko w `index.html`. Dodaj `styles.css`/`app.js` jak zaczną się komplikacje.
